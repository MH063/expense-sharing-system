## 一、用户认证与账户管理接口 (user.js)
1. 登录接口 - POST /auth/login
2. 注册接口 - POST /auth/register
3. 登出接口 - POST /auth/logout
4. 刷新令牌接口 - POST /auth/refresh-token
5. 获取当前用户信息 - GET /users/profile
6. 更新用户资料 - PUT /users/profile
7. 上传用户头像 - POST /users/avatar
8. 获取用户通知 - GET /notifications
9. 标记通知为已读 - PUT /notifications/{id}/read
10. 批量标记通知为已读 - PUT /notifications/mark-all-read
11. 删除通知 - DELETE /notifications/{id}
12. 获取用户设置 - GET /users/settings
13. 更新用户设置 - PUT /users/settings
14. 绑定第三方账号 - POST /users/bind-third-party
15. 解绑第三方账号 - DELETE /users/unbind-third-party
16. 获取用户活动日志 - GET /users/activity-logs
